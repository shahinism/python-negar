Negar

Negar trying to be a spell corrector for persian language.

Welcome to Negar Pages.

Negar will not do any special thing except try to help you in write a more beautiful Persian text. for now it helps you in editing your pages with legal space character & complete Persian alphabet. In the future it will have a spell corrector engine and a graphical user interface (GUI).

Screenshot

How to install:

$ git clone https://github.com/shahinism/Negar.git
$ cd Negar
$ sudo python setup.py install

how to use it:

$ Negar.py [ARGUMENTS] -f [INPUT_FILENAME] -o [OUTPUT_FILENAME]

Arguments are:

-h, --help                       Display this help and exit
-V, --Version                    Print version number and exit
-f, --file[=INPUT_FILE_NAME]     Specify [INPUT_FILE]. The file who you want to edit
-o, --output[=OUTPUT_FILE_NAME]  Specify [OUT_PUT_FILE]. The file who you want the programs
                                 output writes into it. If you don't specify this option
                                 Negar will generate an auto file to save the result.
